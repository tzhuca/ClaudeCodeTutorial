# 第一章：认识 Claude Code

## 💡 一句话理解

Claude Code 是一个运行在命令行中的 AI 编程助手，通过对话即可完成代码编写、修改与自动化操作。

---

## 📌 工作方式

传统开发：
- 查资料 → 写代码 → 调试 → 修复

Claude Code：
- 描述需求 → AI执行 → 自动修改 → 输出结果

---

## 🧠 思维转变（非常重要）

使用 Claude Code 的核心不是“写代码”，而是：

> **表达需求的能力**

# 第二章：基础术语

| 术语 | 含义 | 示例 |
|------|------|------|
| CLI | 命令行界面 | 终端 |
| REPL | 交互循环 | 持续对话 |
| Token | 计费单位 | 输入输出文本 |
| Slash 命令 | /开头命令 | /help |
| Checkpoint | 状态快照 | 存档 |
| Rewind | 回退 | 读档 |

---

# 第三章：三种模式（核心）

## 1️⃣ 交互模式（最重要）

```bash
claude
```

特点：
- 多轮对话
- 上下文记忆
- 适合复杂任务

---

## 2️⃣ 单次执行模式

```bash
claude "写一个快速排序"
```

特点：
- 一次执行
- 无上下文

---

## 3️⃣ 打印模式（自动化）

```bash
claude -p "生成JSON"
```

特点：
- 纯文本输出
- 适合脚本/管道

# 第七章：Slash 命令（高频）

## 基础命令

| 命令 | 作用 |
|------|------|
| /help | 查看帮助 |
| /exit | 退出 |
| /clear | 清空 |
| /compact | 压缩上下文 |

---

## 上下文管理

| 命令 | 作用 |
|------|------|
| /context | 查看Token |
| /cost | 查看费用 |

---

## 会话管理

| 命令 | 作用 |
|------|------|
| /resume | 恢复 |
| /export | 导出 |
| /rename | 重命名 |

# 第八章：效率技巧
